Bong

  • Category: Software Development
  • Language: Python, Java, Ivy
  • Project date: march 2022
  • Github Repository: Bong

Introduction

For a school project, we had to create a search engine for documents. We use C to code the engine and Java for the interface! To make the communication between the engine and the interface, we used Ivy (a software bus). For this project, I worked on search algorithms and the communication betwen the motors and the interface.

Structure

The program is divided into 2 parts : The Interface and search engines. Both communicate throught the software bus ivy by following a protocol that we have defined.

The user select what types of files he wants to search and compare (text, picture or audio). Then, he can write key words and ban words. When he launches the search, a loading screen appears and the interface sends a request to engines. Depending on the settings, a motor will receive this request and answered with a list of files. The interface receives this list and displays it to the user who can click on each file to open it.

An administrator can access to settings and modify: which motors will answer (one of them or both), the minimum accepted resemblance rate and advanced engine settings...
