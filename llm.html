Creation of a generative AI system based on Carl-Gustav Jung's theory of the 16 personalities

  • Category: Games and Generative AI
  • Language: Blueprint and C++ and Python
  • Date: Still in progress

Introduction

The main aim of this project is to explore the possibility of using LLMs to create an original system for generating and evolving antagonistic NPCs in an open-world game. Based on Carl-Gustav Jung's theory of personalities, this system relies on the analysis of the player's actions and decisions to evolve their characteristics and those of their opponents by generating random events between NPCs (friendly interaction, combat, ambush, etc.)
