  • Category: Computer vision
  • Language: Python (Sklearn, Pygame)
  • Project date: October 2022
  • Github Repository: Calculator

Introduction

It's a little project to apply concepts of Neural Network, Image classification and detection that I learned. The objective is to create an application which recognize digits. Results are satisfactory and I will push the application a little further by trying to solve simple calculations.

The most important thing is Data !

If there's one thing I've come to understand about machine learning, it's that the most important thing is the data! So I trained my model with the famous MNIST dataset. But, each number is a 28x28 matrix. Therefore, I have transformed the user's drawing (RGB, 280x280) into a 28x28 BW matrix to give it to the model.
